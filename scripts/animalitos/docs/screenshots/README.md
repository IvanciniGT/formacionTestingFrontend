# Screenshots

Este directorio contiene las capturas de pantalla de la aplicación Animalitos.

Para generar nuevas capturas:

1. Ejecutar `npm run dev`
2. Abrir http://localhost:4200
3. Capturar pantallas de:
   - Lista principal
   - Vista de detalle
   - Formulario de creación/edición

## Capturas Actuales

- `lista.png` - Vista principal con grid de animalitos
- `detalle.png` - Vista detallada de un animalito
- `formulario.png` - Formulario de creación/edición

## Dimensiones Recomendadas

- Ancho: 1200px
- Alto: Variable según contenido
- Formato: PNG para mejor calidad
